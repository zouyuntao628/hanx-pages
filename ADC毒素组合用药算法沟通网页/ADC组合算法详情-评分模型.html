组合推荐算法评分模型

本页解释算法如何把“机制不同且互补”“毒性要分开”“有效区间要接近”这三类业务语言转化为可计算、可解释、可迭代的评分系统。

1. 总体思路

早期数据有限，不建议一开始就做完全黑箱的深度学习模型。更稳妥的 MVP 是“规则模型 + 知识图谱 + 证据权重”，让每一个组合推荐都能解释为什么推荐、为什么不推荐、哪些风险需要实验验证。

Combination Score = 0.30 * Mechanism Complementarity + 0.20 * Resistance Pathway Blockade + 0.15 * Biomarker Selectivity + 0.15 * Dose Window Compatibility + 0.10 * Clinical / Experimental Feasibility + 0.10 * Novelty - 0.25 * Toxicity Overlap Penalty - 0.15 * Mechanism Redundancy Penalty - 0.10 * PK Incompatibility Penalty
权重不是固定不变的。若甲方目标是“发现强协同”，机制互补权重可提高；若目标是“临床转化”，毒性惩罚和临床可行性权重应提高。

2. 机制互补分

高分关系

  • 一个药造成 DNA 损伤，另一个药阻断 DNA repair。
  • 一个药造成复制压力，另一个药阻断 ATR/CHK1/WEE1 checkpoint。
  • 一个药诱导有丝分裂压力，另一个药增强凋亡执行。
  • 一个药攻击主路径，另一个药阻断已知耐药替代路径。

低分或惩罚关系

  • 两个药打同一个靶点，机制高度重复。
  • 两个药都主要依赖同一耐药敏感因素，例如同为强 P-gp 底物。
  • 药效可能增强，但正常组织毒性也同方向增强。
  • 缺少可解释的细胞过程连接，只是文献中偶然共现。
机制关系 示例 评分倾向
损伤 + 修复阻断 Topo I inhibitor + PARP / ATR / DNA-PK inhibitor 4-5 分
复制压力 + checkpoint 解除 Exatecan + WEE1 / CHK1 inhibitor 4-5 分
有丝分裂压力 + 凋亡启动 MMAE + BCL2/MCL1 axis inhibitor 3-4 分
同类毒素叠加 Topo I inhibitor + Topo I inhibitor 0-1 分，并触发重复惩罚

3. 毒性重叠惩罚

客户强调“安全性角度，药物毒性要分开”。这可以转化为毒性向量重叠计算。理想组合不是完全没有毒性，而是药效互补的同时，主要剂量限制毒性不要高度集中在同一器官系统。 

toxicity_overlap_penalty =
  sum(min(toxicity_A[organ], toxicity_B[organ]) * organ_weight[organ])

organ_weight 可根据项目目标调整：
bone_marrow = 高权重
cardio = 高权重
neuro = 高权重
liver/kidney = 中高权重
GI/skin = 中等权重，视药物类别而定
毒性组合 判断 算法处理
骨髓抑制 + 骨髓抑制 高风险，尤其是 DNA damage 药物组合 提高惩罚，建议低剂量矩阵和正常细胞对照
神经毒性 + 神经毒性 高风险，微管毒素组合尤其需要谨慎 除非机制优势明显，否则降低优先级
不同 DLT，且有效区间接近 安全性适配较好 保留高优先级，进入实验验证

4. 剂量窗口匹配

“不能一个 1000 起效果，一个 1 起效果”对应的是有效浓度窗口不匹配。算法要比较体外 potency，也要比较临床可达暴露。 

potency_ratio = max(IC50_A, IC50_B) / min(IC50_A, IC50_B)
exposure_margin = free_Cmax_or_AUC / IC50_or_EC50

if potency_ratio < 10:
    dose_score = 5
elif potency_ratio < 100:
    dose_score = 4
elif potency_ratio < 1000:
    dose_score = 2
else:
    dose_score = 0 or 1
ADC payload 场景要特别谨慎：payload 的体外 IC50、ADC 递送后的细胞内浓度、血浆游离 payload 暴露和 bystander effect 暴露不是同一个概念，需要分字段记录。

5. 推荐输出解释模板

推荐组合：Exatecan + ATR inhibitor
综合结论：高优先级
推荐理由：
  - Exatecan 造成复制压力和 DNA 损伤。
  - ATR 抑制削弱复制压力应答。
  - 两者属于“损伤诱导 + 损伤耐受阻断”的互补关系。
风险：
  - 骨髓抑制和正常增殖组织毒性可能叠加。
适用背景：
  - SLFN11 high、TP53 mutation、高复制压力、DNA repair deficient。
实验建议：
  - 8x8 dose matrix。
  - Bliss / Loewe / ZIP / HSA 四种 synergy score。
  - 检测 gamma-H2AX、apoptosis、cell cycle。
