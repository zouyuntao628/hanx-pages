双载药 [毒+免] ADC + 外源口服 [控] 方案开发平台

本页把“细胞毒 payload + 免疫激动 payload 的双载药 ADC，再联用外源口服调控小分子”的思路整理成可执行方案。定位不是马上承诺一个固定三联临床方案，而是建立一个可以由学生先做干实验、再交接给湿实验部门做小规模高通量组合矩阵验证的平台。

主线：毒+免双载药 ADC 增强模块：外源口服调控药物 核心风险：CRS / DAR / 骨髓毒性 首个样例：Exatecan + STING + ATRi/WEE1i/PARPi

一、方案结论

该方向具备科学合理性和产业可行性，但应平台化、分阶段推进。第一阶段聚焦 [Topo I payload + STING agonist] 双载药 ADC 本身；第二阶段再把 ATRi、WEE1i、PARPi、PD-1/PD-L1、myeloid modulator 等外源调控药物作为候选增强模块筛选。ATRi 不宜一开始绑定成固定三联方案。
模块 可行性 风险 建议定位
[毒+免] 双载药 ADC 高 中高 作为主线平台，优先验证 payload ratio、linker 稳定性和肿瘤局部免疫激活。
Exatecan / Topo I + STING 高 中高 机制自洽：DNA damage + antigen release + innate immune activation。
外源口服 ATRi 中 高 作为增强模块筛选，不作为初始固定组合。重点评估骨髓毒性和给药时序。
小规模高通量组合矩阵 高 中 适合由干实验先筛候选，再交给湿实验部门做 6x6/8x8 验证。

二、公开进展与证据等级

产业进展会持续变化，正式对外报告建议把证据分成“公开可核验”“公开临床前资料”“推断/待核验”三类，避免把尚未公开验证的结论写死。

项目/方向 公开信息 证据等级 对本方案的意义
Astellas / Sutro ASP2998 TROP2 靶向 dual-payload iADC，公开资料显示为 Topoisomerase I inhibitor + STING agonist，并已进入首次人体研究。 公开可核验 说明 [Topo I + STING] 双载药 iADC 已经从概念走向早期临床验证。
Sutro HER2 iADC 公开报告中披露 exatecan + STING agonist 双载药 HER2 iADC 的临床前疗效和探索性 NHP 安全性设计。 公开临床前资料 支持 HER2 场景下 [毒+免] 双载药 ADC 的可行性和工程设计思路。
Daiichi Sankyo DS3610 STING agonist ADC 进入 I 期临床；不是明确的 [Topo I + STING] 双载药 ADC。 公开可核验 说明 STING agonist 作为 ADC 免疫 payload 正在被临床验证。
ATRi 与 Topo I/化疗联用 机制上合理，但已有研究提示 ATRi 与 DNA replication-targeting 化疗联用可能存在显著骨髓毒性。 公开临床经验 ATRi 应作为高风险增强模块，必须做剂量和时序优化。
建议在甲方材料中使用“该方向已进入早期产业验证阶段”这样的表述，而不是笼统写“多款已完成 NHP 并提交 IND”。后者除非有明确项目和来源，否则容易被追问。

三、机制链路

1. 毒素 payload

Exatecan / DXd 类 Topo I payload 诱导 DNA damage、replication stress 和 tumor cell death，同时释放抗原和 damage-associated signals。

2. 免疫 payload

STING agonist 激活 innate immunity、type I IFN signaling、抗原呈递和 T cell recruitment，目标是把冷肿瘤局部转热。

3. 外源控制药物

ATRi/WEE1i/PARPi/PD-1 等不是固定答案，而是候选增强模块，用于阻断修复、增强免疫或克服耐药。

理想作用链路：
Topo I payload
=> DNA damage + replication stress
=> tumor antigen release + immunogenic cell death signal

STING agonist payload
=> local innate immune activation
=> type I IFN / CXCL10 / antigen presentation
=> T cell recruitment and immune memory

External oral modulator
=> blocks DNA damage tolerance or resistance pathway
=> expands efficacy window if toxicity remains manageable

四、关键术语解释

这一节给学生先看。下面列出本方案最核心的一组术语，完整版本请查看独立页面： ADC 组合算法术语字典。

术语 中文解释 在本方案中的作用 学生理解要点
TROP2 肿瘤相关细胞表面抗原，全称 trophoblast cell-surface antigen 2。 作为 ADC 的“导航地址”。如果肿瘤细胞表面 TROP2 高表达，抗体可以更容易把 payload 送进去。 把它理解成肿瘤细胞表面的门牌号。门牌号越清楚，ADC 越容易找到目标。
HER2 人表皮生长因子受体 2，是乳腺癌、胃癌等肿瘤中的重要靶点。 也是 ADC 常用靶点。Enhertu 就是 HER2 ADC 的代表。 HER2 是成熟靶点，有很多已有经验，适合作为方案验证靶点。
Topo I Topoisomerase I，拓扑异构酶 I，是 DNA 复制和转录时负责缓解 DNA 扭曲的酶。 Topo I payload 抑制这个酶，导致 DNA 损伤和复制压力，最终杀死肿瘤细胞。 把 Topo I 理解成 DNA 复制时的“解结工具”。抑制它后，DNA 容易断裂，细胞更容易死亡。
Topo I payload 以 Topo I 为作用机制的 ADC 毒素载荷，例如 exatecan、DXd、SN-38 类。 作为 [毒+免] 双载药 ADC 中的“毒”模块，负责直接杀伤肿瘤细胞。 这是 ADC 上挂的细胞毒药物，不是普通游离小分子给药。
STING Stimulator of Interferon Genes，先天免疫通路中的关键蛋白。 STING 被激活后可诱导 type I interferon、CXCL10 等信号，促进抗原呈递和免疫细胞进入肿瘤。 把 STING 理解成免疫系统的报警器。报警器在肿瘤局部响是好事，在全身乱响就可能引发炎症风险。
STING agonist STING 激动剂，可以激活 STING 通路的小分子或 payload。 作为双载药 ADC 中的“免疫”模块，目标是把冷肿瘤局部变热。 最大风险是游离 STING agonist 在血液中释放后引发 cytokine release 或 CRS。
ATRi ATR inhibitor，ATR 抑制剂。ATR 是细胞应对复制压力和 DNA 损伤的重要 checkpoint kinase。 与 Topo I payload 联用时，可能阻断肿瘤细胞修复 DNA 损伤，从而增强杀伤。 机制强，但骨髓毒性风险也高。不能默认安全，必须低剂量和时序验证。
WEE1i WEE1 inhibitor，WEE1 抑制剂。WEE1 参与 G2/M 细胞周期 checkpoint。 让带有 DNA 损伤的肿瘤细胞更难停下来修复，可能促使其进入有丝分裂灾难。 特别适合考虑 TP53 突变、高复制压力、高增殖模型。
PARPi PARP inhibitor，PARP 抑制剂。PARP 参与 DNA 单链损伤修复。 与 Topo I payload 组合时，可能进一步削弱 DNA 修复能力，尤其适合 HRD/BRCA 缺陷背景。 要注意 HRD、BRCA、SLFN11 等 biomarker 分层，不能所有细胞系一概而论。
CHK1i CHK1 inhibitor，CHK1 抑制剂。CHK1 是 ATR 下游的 checkpoint 调控节点。 阻断 DNA damage checkpoint，减少肿瘤细胞修复窗口。 和 ATRi/WEE1i 类似，可能增强杀伤，也可能放大正常增殖组织毒性。
DNA-PKi DNA-PK inhibitor，DNA-PK 抑制剂。DNA-PK 参与 DNA 双链断裂修复，尤其是 NHEJ 修复通路。 用于探索 DNA repair blockade，与 DNA 损伤类 payload 可能形成组合增益。 这是修复阻断方向，学生要关注是否会与 Topo I payload 产生毒性叠加。
PD-1 / PD-L1 免疫检查点通路。PD-1 在 T 细胞上，PD-L1 常在肿瘤或免疫抑制细胞上。 如果 STING payload 能把肿瘤变热，PD-1/PD-L1 阻断可能进一步释放 T 细胞抗肿瘤活性。 这类组合更偏免疫增强，不是直接增强 DNA 损伤杀伤。
CSF1R Colony Stimulating Factor 1 Receptor，与肿瘤相关巨噬细胞生存和功能有关。 CSF1R 调控可用于改变肿瘤微环境中的髓系免疫抑制。 这是肿瘤微环境调节方向，适合和 STING 免疫激活逻辑一起评估。
TGF-beta Transforming Growth Factor beta，转化生长因子 beta，是免疫抑制和肿瘤微环境重塑相关通路。 TGF-beta 通路抑制可能解除免疫抑制，使 STING 激活后的免疫反应更有效。 这是免疫微环境方向，重点看是否能改善冷肿瘤和免疫排斥型肿瘤。
CRS Cytokine Release Syndrome，细胞因子释放综合征。 STING agonist 系统性释放后可能引发的严重安全风险。 看到 IL-6、TNFα、IFN 等指标异常升高时，要优先考虑 CRS 风险。
DAR Drug-to-Antibody Ratio，平均每个抗体上挂了多少个 payload。 双载药 ADC 中要分别关心毒素 DAR、免疫 payload DAR 和总 DAR。 DAR 不是越高越好。过高可能导致聚集、PK 变差、毒性增加。
payload ratio 双载药 ADC 中两种 payload 的比例，例如 Topo I payload : STING agonist。 决定药效和安全性平衡，是双载药 ADC 的核心 CMC 参数。 学生要关注是否能稳定控制比例，不能只看平均 DAR。

五、核心风险与控制策略

风险 机制 控制策略 干实验任务
STING 系统性释放导致 CRS 游离 STING agonist 激活外周单核/巨噬细胞，引发 IL-6、TNFα、type I IFN 等升高。 稳定 linker、低 STING DAR、降低 FcγR 摄取、PBMC/whole blood cytokine assay、NHP cytokine monitoring。 整理 STING payload、linker、DAR、cytokine release 文献和竞品策略。
DAR 和 payload ratio 漂移 双 payload 随机偶联可能导致批次间毒素/免疫比例不稳定。 site-specific conjugation、orthogonal chemistry、branched linker、LC-MS/HIC/SEC 质控。 整理双载药 ADC 工艺路线和可质控指标。
外源 ATRi 放大骨髓毒性 Topo I payload 造成 DNA damage，ATRi 阻断修复和复制压力应答，正常增殖组织也可能受损。 低剂量、短程、间歇给药；优先体外筛选；加入正常细胞/骨髓 proxy；设立 No-Go 标准。 整理 ATRi、WEE1i、PARPi 与 Topo I/ADC 联用证据和毒性谱。
免疫激活发生在外周而非肿瘤 ADC 不稳定或 Fc 介导摄取导致免疫细胞系统性激活。 抗体靶点选择、Fc engineering、linker stability、肿瘤局部 vs 外周血 cytokine 对比。 建立靶点表达、内吞和免疫细胞表达数据库。

六、学生可执行的干实验工作计划

下面的计划按 8 周设计，适合安排 2-4 名学生分工完成。目标不是简单做文献综述，而是形成一个可以交给湿实验部门执行的小规模高通量组合矩阵实验包。每一周都要回答三个问题：为什么做、怎么做、做到什么程度算完成。

学生开始执行前，建议先阅读两个辅助页面： 术语字典 和 学生阅读书目与章节。
第 1 周

定义项目边界和检索词表

为什么做：学生如果一开始不统一术语，很容易把普通 ADC、免疫 ADC、双载药 ADC、ADC 联用小分子混在一起，后面整理出的数据无法比较。

怎么做：先阅读上一节术语表，再把项目拆成 4 个对象：靶点、细胞毒 payload、免疫 payload、外源调控药物。每个对象建立中英文关键词，并规定哪些信息算“可核验证据”。

  • 确定主线：TROP2/HER2 作为 ADC 靶点，Topo I payload 作为细胞毒模块，STING agonist 作为免疫激活模块。
  • 确定外源药物候选类别：DNA 损伤应答抑制剂 ATRi/WEE1i/PARPi/CHK1i/DNA-PKi，以及免疫调控方向 PD-1/PD-L1、CSF1R、TGF-beta 等。
  • 建立检索词表：dual-payload ADC、iADC、STING agonist ADC、exatecan ADC、Topo I ADC、ATR inhibitor ADC combination。
  • 完成标准：形成一张“项目词表 + 候选类别 + 排除范围”表格，导师可以据此判断学生后续检索是否跑偏。
第 2 周

竞品和产业进展调研

为什么做：这个方向属于前沿赛道，很多说法容易被放大。学生要把“已经进入临床”“只有临床前数据”“只是平台能力”区分开。

怎么做：优先查公司官网、临床试验登记、会议 PDF、专利、论文和投资者材料。每条信息都要保留来源链接和发布日期。

  • 整理 ASP2998、Sutro HER2 iADC、DS3610 等公开资料。
  • 记录靶点、payload、linker、DAR、开发阶段、公开疗效和安全性信息。
  • 区分公开可核验信息和推断信息。
  • 完成标准：形成竞品矩阵表，至少包含项目名称、公司、靶点、payload、开发阶段、证据等级、原始来源链接和一句话结论。
第 3 周

payload 和 linker 风险库

为什么做：[毒+免] 双载药 ADC 的成败不只看 payload 是否有效，更看 linker 是否稳定、payload ratio 是否可控、游离 STING 是否会触发全身炎症。

怎么做：把 payload 和 linker 当成工程模块整理。每个 payload 记录作用机制、有效浓度、系统毒性、是否容易外排、是否适合 bystander effect；每个 linker 记录释放机制、血浆稳定性和可质控指标。

  • 整理 Topo I payload：exatecan、DXd、SN-38 等。
  • 整理 STING agonist payload 的活性、释放风险、cytokine 风险。
  • 整理 linker 类型：cleavable、non-cleavable、branched linker、site-specific conjugation。
  • 完成标准：形成 payload/linker 风险评分表，至少包括 efficacy、CRS risk、myelotoxicity risk、DAR feasibility、linker stability、evidence level 六个字段。
第 4 周

外源口服调控药物库

为什么做：ATRi 机制很强，但风险也高。不能只盯 ATRi，需要把 WEE1i、PARPi、CHK1i、DNA-PKi、PD-1/PD-L1 等作为可替代模块比较。

怎么做：每类药物选择 5-10 个代表分子，优先临床阶段或已上市药物。记录机制、毒性、临床暴露、是否有 ADC 或 Topo I 联用证据。

  • 每类药整理 5-10 个代表分子，优先上市/临床阶段药物。
  • 记录靶点、机制、临床阶段、主要毒性、IC50/暴露、可购买性。
  • 标注是否适合增强杀伤、克服耐药、降低剂量或免疫增强。
  • 完成标准：形成外源药物候选库，并给出每个药的推荐用途：杀伤增强、耐药阻断、免疫增强、安全性探索或不推荐。
第 5 周

细胞系和 biomarker 筛选

为什么做：组合药效强弱高度依赖细胞背景。没有 biomarker 分层，即使实验做出协同，也很难解释为什么有效、适合哪个癌种。

怎么做：围绕靶点表达和机制相关 biomarker 建立细胞系面板。至少覆盖靶点高低表达、SLFN11 高低、TP53 突变/野生型、HRD/BRCA 状态、ABCB1 高低和 STING pathway competence。

  • 按靶点表达选择 TROP2/HER2 高低表达细胞系。
  • 整理 SLFN11、TP53、HRD/BRCA、ABCB1、STING pathway competence 等 biomarker。
  • 区分 exatecan sensitive 和 resistant 模型。
  • 完成标准：形成 6-8 个肿瘤细胞系 + 1-2 个安全性 proxy 模型的推荐清单，并说明每个模型在实验中的用途。
第 6 周

组合评分和 Top 候选输出

为什么做：湿实验资源有限，不能把所有组合都做一遍。干实验的核心价值是把候选组合从几十个收敛到 Top 3-6 个。

怎么做：用统一评分表评估每个组合。评分维度包括机制互补、耐药路径阻断、毒性重叠、剂量窗口、临床/实验可行性和证据等级。

  • 按机制互补、耐药阻断、毒性重叠、剂量窗口、实验可行性打分。
  • 输出 Top 10 外源调控药物，进一步收敛到 Top 3-6 个湿实验优先组合。
  • 每个组合写出推荐理由、风险和实验读出。
  • 完成标准：形成组合排序表和单组合解释卡片。每张卡片必须包含推荐理由、风险、适合细胞系、建议剂量矩阵和预期读出。
第 7 周

湿实验交接包设计

为什么做：湿实验部门需要的是可以直接排板和执行的实验包，而不是泛泛的“建议做实验”。这一周要把干实验结果翻译成实验参数。

怎么做：为每个组合定义药物、细胞系、剂量范围、矩阵大小、重复数、读出指标、时间点和数据回填格式。

  • 确定每个组合的剂量范围建议和是否使用 6x6 或 8x8 矩阵。
  • 定义读出指标：viability、Bliss/Loewe/ZIP/HSA、gamma-H2AX、IFN-beta、CXCL10、IL-6、TNFα、apoptosis。
  • 加入正常细胞或 PBMC cytokine 风险评估建议。
  • 完成标准：形成湿实验设计表、样本命名规则、排板逻辑、数据回填模板和异常结果记录规则。
第 8 周

形成汇报材料和决策包

为什么做：最终结果要同时服务甲方、导师和湿实验部门。甲方看项目价值，导师看逻辑闭环，湿实验部门看可执行性。

怎么做：把前 7 周的表格和结论整理成三套材料：甲方汇报版、内部技术版、湿实验执行版。

  • 整理面向甲方的 10-15 页汇报。
  • 整理面向湿实验部门的 SOP 风格实验包。
  • 整理 Go/No-Go 判断标准。
  • 完成标准：交付 PPT、Excel 数据包、湿实验 SOP 草案、Top 3-6 组合推荐卡和 Go/No-Go 判断表。

学生 D：细胞系和实验设计 整理靶点表达、biomarker、细胞系、读出指标和组合矩阵方案。 细胞系推荐表、实验矩阵表、数据回填模板。

八、哪些任务可以用人工智能辅助完成

人工智能适合做信息检索、结构化整理、初步归纳、表格生成和报告草拟，但不应替代学生做证据核验和科学判断。所有 AI 输出都要回到原始文献、公司公告、临床登记或数据库进行确认。

任务 AI 可以做什么 学生必须人工核验什么 建议输出
检索词扩展 根据“dual-payload ADC / STING agonist ADC / exatecan ADC”扩展中英文检索词、同义词、公司名、药物名和靶点名。 确认关键词是否真的属于本项目范围，删除无关方向，例如普通免疫联合或非 ADC 药物。 检索词表、排除词表、数据库检索策略。
文献和网页初筛 快速浏览标题、摘要、公司新闻稿和会议材料，提取项目名称、靶点、payload、阶段和一句话结论。 打开原文确认是否准确，尤其是开发阶段、payload 类型、是否真的是双载药 ADC。 文献初筛表、证据等级标记。
竞品矩阵生成 把公开资料整理成统一表格，包括公司、项目、靶点、payload、linker、DAR、阶段、优势和风险。 核对来源链接和日期；把“推断信息”和“公开披露信息”分开。 竞品矩阵 Excel、证据来源索引。

外源药物候选库 批量整理 ATRi、WEE1i、PARPi、CHK1i、DNA-PKi 等候选药物的靶点、机制、临床阶段和主要毒性。 核对具体分子是否可购买、是否临床可用、是否有严重毒性警告、是否适合体外实验。 外源药物候选库和初步排名。
机制链路绘制 把“Topo I damage + STING activation + ATRi blockade”转成机制链路、流程图说明和组合假设。 确认机制不是过度推断；每条关键机制至少对应一条证据来源。 机制图、组合假设卡片。
组合评分初稿 根据预设评分维度给候选组合做初步打分，并生成推荐理由和风险提示。 人工检查评分是否被单篇文献带偏；关键组合需要导师复核。 Top 10 排序表、Top 3-6 推荐卡。
细胞系筛选辅助 根据 CCLE/DepMap/文献中的靶点表达和 biomarker 信息，初步推荐细胞系组合。 确认实验室是否能获得细胞系，是否有培养条件，biomarker 数据是否可靠。 细胞系面板和分层理由。
实验矩阵草案 根据候选组合、细胞系和剂量范围生成 6x6/8x8 矩阵草案、样本命名规则和数据回填模板。 剂量范围必须由单药 IC50 或预实验确认，AI 不能直接决定最终剂量。 湿实验排板草案、数据模板。
汇报材料草拟 把表格和结论整理成甲方汇报、内部讨论稿和湿实验交接说明。 检查每个结论是否有来源、是否过度承诺、是否混淆“临床前”和“临床”。 PPT 初稿、执行摘要、风险清单。
建议给学生规定一条硬规则：AI 生成的每一个关键事实都必须附原始来源链接；没有来源的内容只能作为假设或待核验信息，不能进入最终结论。

九、交给湿实验部门的小规模高通量组合矩阵方案

这个实验包的目标不是一次性证明临床可行，而是快速回答三个问题：是否有协同杀伤、是否有免疫激活增益、是否存在明显安全性风险信号。

推荐第一批实验规模

模块 建议规模 说明
ADC 模型 Topo I ADC、STING ADC、Topo I+STING 双载药 ADC、抗体对照 用于判断双载药是否优于单 payload 或简单叠加。
外源药物 Top 3-6 个 优先 ATRi、WEE1i、PARPi，再根据甲方资源加入 PD-1/myeloid modulator。
细胞系 6-8 个肿瘤细胞系 + 1-2 个正常细胞或 PBMC 风险模型 覆盖 TROP2/HER2 高低表达、SLFN11、TP53、HRD、ABCB1 差异。
剂量矩阵 6x6 筛选，重点组合 8x8 复测 先宽筛，再对高协同组合做更细矩阵。
重复数 n=2 初筛，n=3 复测 初筛节省成本，复测保证稳定性。

读出指标

读出 指标 用途
药效 cell viability、IC50 shift、growth inhibition 判断组合是否增强杀伤。
协同 Bliss、Loewe、ZIP、HSA 从不同数学假设计算协同分数。
DNA damage gamma-H2AX、p-RPA、p-CHK1 验证 Topo I payload 和 DDRi 的机制链路。
免疫激活 IFN-beta、CXCL10、ISG signature、STING pathway markers 验证 STING payload 是否产生局部免疫激活。
炎症风险 IL-6、TNFα、IL-1β、IFN-related cytokines 评估游离 STING 或外周免疫激活风险。
安全性 proxy PBMC cytokine release、normal cell viability、骨髓相关细胞活性 初步判断是否存在治疗窗口。

十一、Go / No-Go 标准

Go 标准

  • 双载药 ADC 比单 payload ADC 有明确疗效或免疫激活优势。
  • STING payload 释放风险可控，PBMC cytokine release 不出现危险信号。
  • DAR 和 payload ratio 可被稳定描述和质控。
  • 外源口服调控药物在低剂量或短程给药下扩大药效窗口。
  • 肿瘤模型敏感性明显高于正常细胞或 PBMC 风险模型。

No-Go 标准

  • 轻微 STING 释放即引发强 cytokine release。
  • 双载药 ADC 不优于单 payload ADC 或混合给药。
  • payload ratio 无法稳定控制，批次异质性过大。
  • ATRi/WEE1i 只放大毒性，不增加治疗窗口。
  • 正常细胞、PBMC 或骨髓 proxy 比肿瘤细胞更敏感。
